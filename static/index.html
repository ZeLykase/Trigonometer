Теоретические основы тригонометрии

Основные понятия

Тригонометрия — раздел математики, изучающий соотношения между сторонами и углами треугольников и тригонометрические функции.

Тригонометрические функции

Для острого угла прямоугольного треугольника:

sin(α) = противолежащий катет / гипотенуза
cos(α) = прилежащий катет / гипотенуза
tg(α) = противолежащий катет / прилежащий катет
ctg(α) = прилежащий катет / противолежащий катет

Тригонометрическая окружность

Единичная окружность с центром в начале координат позволяет определить тригонометрические функции для любых углов.

x = cos(α), y = sin(α)

Основные тригонометрические тождества

sin²(α) + cos²(α) = 1
1 + tg²(α) = 1/cos²(α)
1 + ctg²(α) = 1/sin²(α)
tg(α) = sin(α)/cos(α)
ctg(α) = cos(α)/sin(α)

Формулы сложения

sin(α ± β) = sin(α)cos(β) ± cos(α)sin(β)
cos(α ± β) = cos(α)cos(β) ∓ sin(α)sin(β)
tg(α ± β) = (tg(α) ± tg(β)) / (1 ∓ tg(α)tg(β))

Формулы двойного и половинного угла

sin(2α) = 2sin(α)cos(α)
cos(2α) = cos²(α) - sin²(α) = 2cos²(α) - 1 = 1 - 2sin²(α)
tg(2α) = 2tg(α) / (1 - tg²(α))
sin(α/2) = ±√((1 - cos(α))/2)
cos(α/2) = ±√((1 + cos(α))/2)
tg(α/2) = (1 - cos(α)) / sin(α) = sin(α) / (1 + cos(α))

Формулы преобразования суммы в произведение

sin(α) + sin(β) = 2sin((α+β)/2)cos((α-β)/2)
sin(α) - sin(β) = 2cos((α+β)/2)sin((α-β)/2)
cos(α) + cos(β) = 2cos((α+β)/2)cos((α-β)/2)
cos(α) - cos(β) = -2sin((α+β)/2)sin((α-β)/2)
